sphinx-quickstart - создание нового сфинкс-проекта
make html - запуск сборки

sphinx-apidoc -o __source_sphinx-apidoc ../    -  автоматическое создание
документации.
1-й аргумент, после -o, где создать rst файлы,
2-й аргумент, путь до проекта,
автоматически подтягиваются модули, затем index.rst нужно созлдать вручную
и выпонить make html
специально расположил выходные файлы вне source, нужно исправить выдаваемые
ошибки при импорте нкоторых модулей
